# Trabalho Grafos 

Trabalho de calculo numérico - IFCE 2019.1

## Pre-requisitos

Instalar [meson e ninja](https://mesonbuild.com/Getting-meson.html) para buildar o projeto. Ou use o bom e velho gcc, é só um arquivo mesmo.

## Como usar

```bash
$ meson build
$ cd build
$ ninja
$ ./src/main
```
